# 像素塔防 · Python 版

使用 **Pygame + NumPy** 完整复刻的 Python 版本，功能和 HTML 版完全一致。

## 项目结构

```
pygame_version/
├── main.py      # 游戏入口
├── game.py      # 游戏主循环、UI、事件处理
├── entities.py  # 敌人、塔、子弹、粒子
├── audio.py     # 8-bit 合成音效 + BGM
├── config.py    # 常量、10 张地图、塔定义、难度配置
└── README.md    # 本文件
```

## 运行环境

- Python 3.8+
- `pygame`
- `numpy`

## 安装依赖

```bash
pip install pygame numpy
```

## 启动游戏

```bash
cd pygame_version
python main.py
```

## 操作说明

- **底部塔卡片**：点击选择要建造的塔类型。
- **点击绿色草地**：放置防御塔。
- **点击已放置的塔**：显示白色攻击范围圈，以及：
  - **左上角红色 ×**：拆除该塔，返还 50% 金币。
  - **右上角金色 ↑**：升级该塔（伤害 +28%、射程 +10%、射速 +7%）。
- **开始下一波**：点击 UI 上的红色按钮出怪。
- **⚙️ 设置按钮**：调节音乐音量、切换难度（影响敌人强度和塔位上限）。

## 核心特性（与 HTML 版完全一致）

- ✅ **10 个关卡，10 张不同地图**
- ✅ **4 种塔**：机枪、狙击、冰霜、爆破
- ✅ **升级系统**
- ✅ **拆塔返金币**
- ✅ **每 5 的倍数波出现 BOSS**
- ✅ **难度系统**：简单（无塔位限制）/ 普通（22 塔位）/ 困难（14 塔位）
- ✅ **代码合成的 8-bit 音效和 BGM**（无需任何外部音频文件）
